# Invitation Challenge

This repository contains a program to distribute invites to players in a CTF-like way. Users will need to solve all these questions to know the pickup location, time of pickup, and some extra information regarding the event.

> **Note:** Use the command `sudo ufw allow 1941/tcp` to allow `1941` port on ubuntu's firwall.

> **Note:** If stuck, players can ask for a hint code, which will unlock the hint for the given challenge.

## Challenge 0 - Connect to TCP Server

Once the application is run using ubuntu's `tcpserver` command on a specific port number, players will connect to this application using the `nc` command.

Command to host the application -

    tcpserver 0 1941 ./a.out

Command to connect to the application -

    nc <EC2's public IP address> 1941

## Challenge 2 - Decrypt AES Encrypted message

In this challenge, user will need to decrypt the cryptic answer of the previous question using the following private key.

```
thisisasecretkey
```

Once decrypted, players are greeted with the pickup location of the event.

## Challenge 3 - Calculate Pickup Time

This challenge will test user's Hindu mythology's knowledge, specifically Hindu time units.

```
30 muhurta - 1 danda
```

In military time, translating to `23:36`.
