# Invitation Challenge

You will need to solve all these questions to know the pickup location, time of pickup, and some extra information for the event.

> **Note:** Use the command `sudo ufw allow 1941/tcp` to allow `1941` port on ubuntu's firwall.

## Challenge 0 - Connect to TCP Server

Users are required to connect to a specific port of the given IP address.

```
김정일
```

It should be Kim Jong-il's birth year.

## Challenge 1 - Decrypt BrainF\*ck code

This challenge will prompt user to decode the following code.

```

Once decoded using any BrainF\*ck compiler online, the user will be prompted with the following cryptic information.

```
FCFBF1lqQOL7ljMVrS0CJDa8Q+yCsb+UWYA36Woh3bCZ8hbENdUW7PY/fyLsfR8d

## Challenge 2 - Decrypt AES Encrypted message

Here, user will need to decrypt the following message using the private key generated in the previous challenge.

```
thisisasecretkey
```

Once decrypted, user will finally know the pickup location.

```
927 E University Dr, Tempe, AZ 85281
```

## Challenge 3 - Calculate Pickup Time

This challenge will test user's hindu mythology's knowledge, specifically its time units.

```
30 muhurta - 1 danda
```

It will translate to `23:36`.
